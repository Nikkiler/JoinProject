Topic 2 - Communications

Definitions

Phishing

An email with a link which will download a virus / take the user to a pharming website.

Pharming

A fake website which tries to get the user to enter their personal details.

Keylogger

Logs key presses and sends them back to the hacker.

Virus

A self replicating program that is designed to delete or damage a file
